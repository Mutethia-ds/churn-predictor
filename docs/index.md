# Customer Churn Predictor

Welcome to the documentation for the Customer Churn Predictor project.

This project predicts whether a telecom customer is likely to cancel
their subscription ("churn"), using an end-to-end pipeline covering data
generation, validation, storage, feature engineering, model training,
serving, and monitoring.

- **Architecture** — how the pieces fit together
